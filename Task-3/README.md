# CodeAlpha Task 3 - Data Visualization

## Objective
Transform raw data into meaningful visualizations to identify trends, patterns, and business insights.

## Dataset
- Dataset: Sample Superstore
- Format: CSV
- Records: 9,994 rows
- Columns: 21

## Tools Used
- Python
- Pandas
- Matplotlib

## Visualization Created

### Sales by Category
A bar chart was created to compare total sales across product categories.

## Steps Performed

1. Imported the dataset using Pandas.
2. Grouped data by Category.
3. Calculated total sales for each category.
4. Created a bar chart using Matplotlib.
5. Analyzed category-wise sales performance.

## Findings

1. Technology category generated the highest sales.
2. Furniture category generated moderate sales.
3. Office Supplies category generated comparatively lower sales.
4. Technology products contributed significantly to overall revenue.
5. Visualizations made category performance easier to understand.

## Output

The chart was saved in the charts folder.
