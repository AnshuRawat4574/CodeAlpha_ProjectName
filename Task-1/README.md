# CodeAlpha Task 1 - Web Scraping

## Objective

To collect data from a website using Python web scraping techniques.

## Tools Used

* Python
* Requests
* BeautifulSoup
* Pandas

## Website

https://quotes.toscrape.com/

## Process

1. Connected to the website using Requests.
2. Parsed HTML content using BeautifulSoup.
3. Extracted quotes and author names.
4. Stored the data in a CSV file.

## Output

* quotes.csv
* Extracted quotes and authors dataset

## Learning Outcomes

* Web Scraping Basics
* HTML Parsing
* Data Collection
* CSV File Creation
