# CodeAlpha Data Analytics Internship

## Overview

This repository contains the completed tasks for the CodeAlpha Data Analytics Internship. The projects demonstrate skills in Web Scraping, Exploratory Data Analysis (EDA), and Data Visualization using Python.

---

## Tasks Completed

### Task 1: Web Scraping

#### Objective
Collect data from a website using Python web scraping techniques.

#### Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas

#### Output
- Scraped quotes and author names from a website
- Saved extracted data into a CSV file

#### Files
- scraping.py
- quotes.csv
- Screenshots

---

### Task 2: Exploratory Data Analysis (EDA)

#### Objective
Explore and analyze the dataset to discover patterns, trends, and insights.

#### Dataset
- Sample Superstore Dataset

#### Tools Used
- Python
- Pandas

#### Key Findings
- Technology category generated the highest sales.
- West region generated the highest profit.
- Some sub-categories showed lower profitability.
- Higher discounts negatively impacted profit.
- Consumer segment contributed the highest sales.

#### Files
- eda_analysis.py
- Sample - Superstore.csv
- README.md

---

### Task 3: Data Visualization

#### Objective
Transform raw data into meaningful visualizations.

#### Tools Used
- Python
- Pandas
- Matplotlib

#### Visualization
- Sales by Category Bar Chart

#### Key Findings
- Technology category achieved the highest sales.
- Visual representation made business insights easier to understand.

#### Files
- visualization.py
- sales_by_category.png
- Sample - Superstore.csv
