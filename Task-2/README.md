# Task 2 - Exploratory Data Analysis (EDA)

## Objective

Perform Exploratory Data Analysis (EDA) on the Superstore dataset to understand the data structure, identify patterns, detect anomalies, and generate business insights.

---

## Dataset

* Dataset Name: Sample - Superstore
* Records: 9994
* Columns: 21
* File Format: CSV

---

## Questions Explored

1. Which category generates the highest sales?
2. Which region earns the highest profit?
3. Which sub-category is making losses?
4. How do discounts affect profit?
5. Which customer segment contributes the most sales?

---

## Data Exploration

### Dataset Information

* Total Rows: 9994
* Total Columns: 21
* No missing values found in the dataset.

### Key Columns Used

* Category
* Sub-Category
* Region
* Segment
* Sales
* Profit
* Discount

---

## Analysis Results

### Highest Sales Category

Technology generated the highest sales with approximately **836,154** in total sales.

### Highest Profit Region

West region generated the highest profit with approximately **108,418** profit.

### Loss-Making Sub-Category

Tables was identified as the major loss-making sub-category.

### Discount vs Profit

A negative correlation (-0.219) was observed between Discount and Profit, indicating that higher discounts tend to reduce profit.

### Highest Sales Segment

Consumer segment generated the highest sales with approximately **1.16 million** sales.

---

## Key Findings

* Technology category generated the highest sales.
* West region earned the highest overall profit.
* Tables sub-category showed the highest losses.
* Discounts negatively impact profit.
* Consumer segment contributed the most sales.
